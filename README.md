# 🌱 Smart Agriculture AI System

## Overview

This project is an AI-powered Smart Agriculture System developed using Machine Learning.

It consists of three intelligent modules:

- Crop Recommendation
- Fertilizer Recommendation
- Irrigation Prediction

The system helps farmers make better decisions based on soil and environmental conditions.

---

## Features

### Crop Recommendation
Predicts the most suitable crop using:

- Soil pH
- Soil Moisture
- Temperature
- Humidity
- Rainfall

### Fertilizer Recommendation
Predicts the best fertilizer using:

- Soil Type
- Soil pH
- Soil Moisture
- Organic Carbon
- Electrical Conductivity
- Nitrogen
- Phosphorus
- Potassium
- Crop Type

### Irrigation Prediction
Predicts whether irrigation is required based on:

- Soil Moisture
- Temperature
- Humidity
- Rainfall
